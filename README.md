# Range Accrual HW2F Project


This project implements a pricing engine for Range Accrual derivatives using the Hull-White 2-Factor (HW2F) model.

## Structure

- `src/`: Core implementation files.
- `notebooks/`: Exploration and demo notebooks.
- `tests/`: Unit tests.
- `data/`: Empty folder for data storage.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Check `notebooks/exploration.ipynb` for examples.
